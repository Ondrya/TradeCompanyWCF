1. Файл базы данных находится в папке App_Data
2. Строка подключения имеет placeholder |DataDirectory| - который в рантайме заменяется на путь к App_Data
3. Макет столбцов скопирован из прошлой работы, автогенерация столбцов по контексту отключена.
4. Внутри формы данные сначала считываются в приватные поля для коллекций, которые уже типизированы и по ключу мы получаем моментальный доступ к объекту без необходимости делать приведение к типу selected object
5. Клиент к сервису вызывается через using для высвобождения памяти
